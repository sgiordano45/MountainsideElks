# Images Directory

Add the following images to this folder:

## Required
- `elks-logo.png` — Elks lodge logo (recommended: 200x200px or larger)
- `favicon.png` — Browser tab icon (recommended: 32x32px or 64x64px)

## Hall Rental Photos
- `hall-1.jpg` through `hall-6.jpg` — Photos of the hall from different angles
  - Main room setup for events
  - Bar area
  - Kitchen (if showing)
  - Dance floor
  - Decorated for an event
  - Exterior/entrance

## Officer Photos (optional)
- `officer-exalted-ruler.jpg`
- `officer-leading-knight.jpg`
- `officer-secretary.jpg`

## Event Photos (optional)
- Various photos from past events for the gallery

## Image Guidelines
1. **Compress images** before uploading — use squoosh.app or similar
2. **Recommended sizes:**
   - Hall photos: 1200px wide
   - Officer headshots: 400x400px
   - Event gallery: 800px wide
3. Use **descriptive filenames** (e.g., `hall-setup-tables.jpg` not `IMG_4523.jpg`)
4. **JPEG** for photos, **PNG** for logos/graphics with transparency
